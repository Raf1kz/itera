## How to Capture

1. Start the dev server: `npm run dev`
2. Navigate to http://localhost:5173
3. Take full-window screenshots of each view
4. Save as PNG files with the names above
5. Ensure screenshots show the new design tokens in action

## Design Highlights to Capture

- **Rounded corners:** All surfaces use 2xl (1rem) radius
- **Color palette:** Teal primary (#14b8a6), amber accents (#f59e0b)
- **Typography:** Inter font with proper spacing
- **Shadows:** Soft, minimal depth
- **Focus states:** Visible ring on all interactive elements
- **Contrast:** High readability with neutral grays
